Instructions

Folder path is the location of the published folder in the server filesystem, and it can be given either as absolute or as relative.

Absolute path is a path that points to the same location on the file system regardless of the working directory, and is usually written in reference to a root directory.

Relative path is relative to the working directory, and in Mollify it is the backend folder.
